# Sun–Jupiter L₂ Halo Orbit — Interactive Demo

Live 3D simulation of periodic halo orbits near the Sun–Jupiter L₂ libration point, built with [three.js](https://threejs.org/).
Visualizes the orbit family, invariant manifolds, and the minimum-ΔV correction strategy.

## Quick start

```bash
# clone this repo, cd into docs/
python3 -m http.server 8765
# open http://localhost:8765/ in your browser
```

Or deploy to GitHub Pages (instructions below).

## GitHub Pages deployment

1. Create a new repo (or use existing). Make sure `docs/` is at the repo root.
2. Push this folder:
   ```bash
   git add docs/
   git commit -m "Add interactive halo-orbit demo"
   git push
   ```
3. On GitHub: **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** / folder: **/docs**
   - Save.
4. Wait ~1 min. Your demo lives at `https://<username>.github.io/<repo>/`.

## Scene cheatsheet

| Visible | What it is |
|---------|------------|
| Large yellow sphere (left) | Sun (m₁, 99.9% of system mass) |
| Brown sphere | Jupiter (m₂) — enlarged ~100× for visibility |
| Small white dots | Lagrange points L₁ (between Sun and Jupiter) and L₂ (anti-Sun side of Jupiter) |
| Color tube around L₂ | Halo orbit — color maps correction cost (green = cheap, red = expensive) |
| Yellow ball + ring | Spacecraft current position |
| Magenta ball (after *inject*) | Perturbed spacecraft |
| Red strands | Wᵘ — unstable manifold, where perturbations grow |
| Green strands | Wˢ — stable manifold, where convergent trajectories come from |
| Dark disk at L₂ | Ecliptic plane reference (z = 0) |
| Small RGB axes at L₂ | Local coordinate axes (red=x, green=y, blue=z) |

## Controls (right panel)

- **A_z slider** — pick a halo from the 6-orbit family. Period and instability λ_u update in real time.
- **t/T slider** — move the spacecraft along the orbit. All readouts update live.
- **Play ▶** — animated traversal.
- **→ t\*** — teleport to the cheapest correction phase.
- **inject** — seed a perturbation of magnitude α₀ along the unstable eigenvector. Watch the magenta sphere diverge under real CR3BP dynamics (RK4 integrated in-browser).
- **correct** — apply minimum-ΔV correction at the current phase. Displays |Δv|, measured ΔC, predicted ΔC = −2·ΔKE, and residual (should be ~10⁻¹⁶).
- **Layer toggles** — Wᵘ, Wˢ, family ensemble, primaries, cost colors.
- **View presets** — front, top, side, iso.

## Physics in one paragraph

You are inside a rotating reference frame locked to Sun and Jupiter; both sit still while the frame (and anything not gravitationally tied to them) spins once every 11.86 years. In this frame there are five equilibria — the Lagrange points. L₁ and L₂ are visible as white dots. Both are **saddles**: along one direction, perturbations grow exponentially (factor λ_u ≈ 1300 per orbital period), along another they decay, and along a third they oscillate. Spacecraft parked near L₂ need active stabilization. The halo orbit is a closed periodic solution that circles L₂ once per period (T ≈ 3.22 in non-dimensional units, about 6 years real time). The minimum-ΔV correction law picks the fuel-cheapest moment in each orbit — this is visualized by the color gradient along the tube.

## Math in one paragraph

Equations are CR3BP with μ = 9.53·10⁻⁴. Periodic orbit is found by differential correction on a two-parameter Newton iteration using the monodromy matrix Φ(T). Floquet decomposition gives eigenvalues {λ_u, 1/λ_u, 1, 1, e^{±iω}}; the unstable pair drives the instability. The left eigenvector w_u projects any perturbation onto the unstable mode: α = w_u^T · δU. Minimum-ΔV correction: Δv\* = −α · L_v(t) / ‖L_v(t)‖² where L(t) = Φ(t,0)^{−T} w_u. Energy bookkeeping: ΔC = −2·ΔKE in the rotating frame, where C = 2Ω − v² is the Jacobi integral.
